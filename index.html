Keyboard Shortcuts

  • Space - Start/Stop scrolling
  • ↑↓ - Manual scroll
  • Home - Reset to top
  • F - Toggle fullscreen
  • E - Toggle edit mode
  • Esc - Cancel edits (when in edit mode)
  • Ctrl+Enter - Save changes (when in edit mode)
  • +/- - Increase/decrease scroll speed
  • Q/W - Decrease/increase zoom
  • ←→ - Previous/Next song
  • A - Add song menu
  • S - Saved songs menu
  • I - Import/Export menu
  • K - Show shortcuts
